| 评审问题 | 修订结果 | 落地文档 |
|---|---|---|

| JSONB 热点字段与唯一性无落地 | 已明确热点字段物理列、表达式索引策略和唯一字段处理规则 | `05`、`06` |

| 验收标准不可执行 | 已改为包含前置条件、步骤、期望结果、通过标准的表格用例 | `12` |
| 安全/并发/一致性/性能测试缺失 | 已补专项测试和性能门禁 | `12` |

## 3. P1/P2 修订吸收

已吸收：

- 采购申请作为独立业务对象，不作为资产创建强制前置。
- 附件上传 multipart、大小、类型、权限规范。
- 导入导出异步任务、行级错误、幂等键。
- Webhook HMAC 签名、投递日志、重试字段。
- 种子数据强制改密和测试租户隔离。
- 报表分页、筛选、性能规则。
- 前端工程规范、组件尺寸建议、关键页低保真结构。
- RLS 作为二期纵深防御预留。
