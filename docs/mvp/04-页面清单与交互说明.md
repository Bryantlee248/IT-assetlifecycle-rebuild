# 04-页面清单与交互说明

## 1. 页面清单

平台侧：

- 登录页。
- 租户管理页。
- 平台审计页。

租户侧：

- 工作台首页。
- 用户管理页。
- 组织管理页。
- 角色权限页。
- 资产类型配置页。
- 字段配置页。
- 表单配置页。
- 列表配置页。
- 查询配置页。
- 资产列表页。
- 资产详情页。
- 资产新增/编辑抽屉。
- 生命周期时间线。
- 审批待办页。
- 审批详情页。
- 软件许可证页。
- 证书/域名页。
- 云资源页。
- 导入导出页。
- 报表看板页。
- 通知中心页。
- 附件列表组件。
- API Token 页。
- Webhook 页。
- 审计日志页。

## 2. 工作台首页

展示：

- 资产总数。
- 待审批数量。
- 即将到期数量。
- 待续费许可证数量。
- 待退役资产数量。
- 最近操作记录。

操作：

- 跳转资产列表。
- 跳转审批待办。
- 跳转到期提醒。

## 3. 资产列表页

筛选：

- 资产类型。
- 资产大类。
- 生命周期状态。
- 关键字。
- 责任人。
- 所属组织。
- 位置。
- 动态查询条件。

列表列：

- 由列表配置决定。
- 无权限字段不显示。
- 固定列：资产编号、资产名称、资产类型、生命周期状态。

操作：

- 新增资产。
- 编辑资产。
- 查看详情。
- 执行生命周期动作。
- 建立资产关系。
- 导出。

## 4. 资产详情页

展示：

- 基础信息。
- 动态字段。
- 生命周期状态。
- 生命周期时间线。
- 资产关系。
- 附件。
- 审计记录。

交互：

- 字段按权限显示或脱敏。
- 当前状态允许的生命周期动作显示为按钮。
- 无权限动作不显示，后端仍必须校验。

## 5. 元数据配置页

资产类型配置：

- 树形展示资产类型。
- 支持新增、编辑、启停。

字段配置：

- 字段编码。
- 字段名称。
- 字段类型。
- 必填。
- 唯一。
- 可搜索。
- 可排序。
- 敏感字段。
- 数据源。

表单配置：

- 分组。
- 每组列数。
- 字段顺序。

列表配置：

- 列字段。
- 宽度。
- 固定列。
- 默认排序。

查询配置：

- 关键字。
- 枚举筛选。
- 日期范围。
- 人员选择。
- 组织树。
- 位置树。

## 6. 审批页面

审批待办页：

- 展示当前用户待审批任务。
- 支持按业务类型、状态、提交人筛选。

审批详情页：

- 展示申请信息。
- 展示关联资产。
- 展示审批节点。
- 支持通过、驳回。
- 驳回必须填写意见。

## 7. 导入导出页

功能：

- 选择资产类型。
- 下载导入模板。
- 上传 Excel。
- 展示导入任务进度。
- 展示行级错误。
- 按当前筛选导出。

导出必须遵守：

- 租户隔离。
- 数据权限。
- 字段权限。

## 8. 报表看板页

看板包括：

- 资产概览。
- 生命周期分布。
- 软件授权使用率。
- 到期提醒。
- 待办任务。
- 证书/域名到期。
- 云资源成本归属。
