# 09-报告生成规则说明

## 1. 报表定位

第一版报表用于运营和验收，不做完整 BI 平台。

报表只读取业务数据，不反向驱动业务流程。

## 2. 报表清单

第一版报表：

- 资产概览。
- 生命周期分布。
- 软件授权使用率。
- 到期提醒。
- 待办任务。
- 云资源成本归属。
- 证书/域名到期。
- 审计查询。

## 3. 资产概览

统计：

- 资产总数。
- 有形资产数量。
- 无形资产数量。
- 按资产类型数量。
- 按组织数量。
- 按责任人数量。

数据来源：

- `assets`
- `asset_types`
- `organizations`

权限：

- 按当前用户数据范围过滤。

## 4. 生命周期分布

统计：

- 各生命周期状态资产数量。
- 各资产类型的状态分布。
- 维修中资产数量。
- 待退役资产数量。

数据来源：

- `assets.lifecycle_status`
- `lifecycle_events`

## 5. 软件授权使用率

统计：

- 授权总数。
- 已分配数量。
- 可用数量。
- 使用率。
- 即将到期许可证。
- 已过期许可证。

数据来源：

- `software_license`
- `license_assignment`

计算：

```text
available_quantity = total_quantity - used_quantity
usage_rate = used_quantity / total_quantity
```

## 6. 到期提醒

提醒对象：

- 维保到期。
- 软件许可证到期。
- 证书到期。
- 域名到期。

默认提醒窗口：

```text
7天
30天
60天
90天
```

## 7. 待办任务报表

统计：

- 待审批任务。
- 待入库资产。
- 待部署资产。
- 待维修完成资产。
- 待续费许可证。
- 待退役资产。

数据来源：

- `approval_tasks`
- `assets`
- `software_license`
- `notifications`

## 8. 云资源成本归属

统计：

- 按云厂商数量。
- 按区域数量。
- 按成本中心数量。
- 按业务系统数量。

第一版仅登记和聚合，不对接云账单。

## 9. 导出规则

所有报表导出必须遵守：

- 租户隔离。
- 数据权限。
- 字段权限。
- 敏感字段脱敏。

无字段权限的数据不能因为导出而泄露。

## 10. 分页、筛选和性能规则

所有明细类报表必须分页：

- 到期提醒。
- 待办任务。
- 生命周期事件明细。
- 许可证分配明细。
- 导入错误明细。

分页参数遵守 `07-API接口规格说明.md`：

```text
page
size
sort
```

筛选参数至少包括：

- 资产类型。
- 生命周期状态。
- 组织。
- 位置。
- 责任人。
- 到期日期范围。
- 创建时间范围。

性能要求：

- 报表查询必须自动注入 `tenant_id`。
- 明细报表不得默认返回全量数据。
- 日期范围查询字段必须有物理列和索引。
- 50,000 条资产规模下常用报表 P95 小于 800ms。
