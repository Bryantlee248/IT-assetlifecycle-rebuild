# 08-模板体系设计

## 1. 模板体系定位

本系统的模板体系不是文档模板，而是元数据驱动体系。

模板包括：

- 资产类型模板。
- 字段模板。
- 表单模板。
- 列表模板。
- 查询模板。
- 生命周期模板。
- 审批模板。
- 导入模板。

## 2. 资产类型模板

第一版内置资产类型：

有形资产：

- 服务器。
- 网络设备。
- 安全设备。

无形资产：

- 软件许可证。
- 证书。
- 域名。
- 云资源。

## 3. 字段模板

字段类型：

```text
text
textarea
number
decimal
date
datetime
enum
multi_enum
bool
user
org
location
asset_relation
file
url
json
```

字段配置项：

- 字段编码。
- 字段名称。
- 字段类型。
- 是否必填。
- 唯一范围。
- 默认值。
- 校验规则。
- 数据源。
- 是否可搜索。
- 是否可排序。
- 是否可见。
- 是否可编辑。
- 是否敏感。
- 是否加密。

## 4. 服务器字段模板

字段：

- `brand` 品牌。
- `model` 型号。
- `sn` 序列号。
- `cpu` CPU。
- `memory_gb` 内存 GB。
- `power_consumption` 功耗。
- `warranty_vendor` 维保厂商。
- `warranty_end_date` 维保到期日。
- `rack` 机柜。
- `u_position` U 位。

## 5. 软件许可证字段模板

字段：

- `vendor` 厂商。
- `license_model` 授权模式。
- `license_key` 授权密钥。
- `total_quantity` 授权总数。
- `start_date` 生效日期。
- `end_date` 到期日期。
- `contract_no` 合同编号。
- `compliance_status` 合规状态。

`license_key` 是敏感字段，默认脱敏。

## 6. 证书/域名字段模板

字段：

- `domain_name` 域名。
- `certificate_subject` 证书主体。
- `expire_date` 到期日期。
- `responsible_user_id` 责任人。
- `business_system` 业务系统。
- `registrar` 注册商。

## 7. 云资源字段模板

字段：

- `provider` 云厂商。
- `region` 区域。
- `resource_id` 资源 ID。
- `resource_type` 资源类型。
- `cost_center` 成本中心。
- `business_system` 业务系统。
- `sync_source` 同步来源。

## 8. 表单模板

表单使用结构化 JSON：

```json
{
  "sections": [
    {
      "title": "基础信息",
      "columns": 2,
      "fields": ["asset_no", "asset_name", "brand", "model"]
    }
  ]
}
```

运行时表单由以下内容合成：

```text
字段定义
表单模板
字段权限
状态权限
当前用户权限
```

## 9. 列表模板

列表模板定义：

- 列字段。
- 宽度。
- 固定列。
- 默认排序。

无权限字段不能出现在列表。

## 10. 查询模板

支持：

- 关键字。
- 枚举。
- 日期范围。
- 用户选择。
- 组织树。
- 位置树。
- 资产类型树。
- 状态选择。
- 数值范围。

## 11. 生命周期模板

机房资产模板：

```text
planned -> purchasing -> inbound -> deployed -> running -> maintenance -> retired -> disposed
```

软件许可证模板：

```text
planned -> purchasing -> available -> assigned -> reclaimed -> renewal_pending -> expired -> retired
```

## 12. 审批模板

第一版支持：

- 指定用户。
- 指定角色。
- 申请人部门负责人。
- 资产负责人。
- 资产所属组织负责人。

不支持：

- 会签。
- 或签。
- 加签。
- 转办。
- BPMN 设计器。
